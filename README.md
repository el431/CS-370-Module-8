•	How do I approach a problem as a computer scientist?
The first step is to thoroughly understand the problem. Identify the core issue, constraints, and desires outcome. Break the problem into smaller, manageable components. Ask questions. This ensures clarity before attempting a solution and prevents wasted effort on incorrect assumptions. Once the problem is decomposed, develop algorithm, step-by-step. Translate the algorithm into code using an appropriate programming language.
